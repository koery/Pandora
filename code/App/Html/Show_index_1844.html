清华附中民族管弦乐音乐会《华韵印象》奏响国家大剧院

发布于：2013年12月21日 已被阅读次

清华附中金帆民乐团表演照

清华大学教授、博士生导师、清华附中校长兼乐团团长王殿军独唱陕北民歌《黄河水手歌》

2011年10月22日上午，清华附中金帆民乐团在国家大剧院成功举行了“青少年普及音乐会”之“华韵印象——青少年民族管弦乐音乐会”。此次音乐会由清华附中民族音乐团的学生乐手和刘文金、杨青、邓建国等中国国乐大师携手向观众展现优秀的中国“非物质文化遗产”。北京市教委体育美育处副处长王军、中国民族管弦乐学会副会长王书伟等领导和来自各界的近1500名观众观看了演出。

音乐会共分为三个部分：

第一部分“幽兰古韵”——展示对中国传统音乐文化的传承，观众欣赏到了有着四千余年历史，由着名古琴演奏大师杨青演奏中国古琴艺术《流水操》和有着六百年的昆曲曲目。身兼清华大学教授、博士生导师的清华附中校长兼乐团团长王殿军独唱陕北民歌《黄河水手歌》把音乐会带向了高潮。

第二部分“海外传音”——观众欣赏到由乐团指挥胡军编创、乐团独创的中国传统音乐与西方音乐合璧的《查尔达什》。由乐团中最小的弦乐乐器高胡和西洋乐队中最庞大的弦乐乐器低音提琴共同演奏。在音乐会现场的演奏中，两位学生演奏员表现的自信、从容、富有激情，完美地表现出高胡的婉约柔美和低音提琴的浑厚宽广；

第三部分“华乐颂祖国”，由着名的作曲家、指挥家刘文金的作品《泰山魂》和《长城随想曲》组成。《泰山魂》表现了泰山的雄奇博大，同时通过和声、配器等手法表达了委婉细腻的感情。同是刘文金的作品《长城随想》，由着名二胡演奏家邓建栋和清华附中金帆民乐团原学生团长、乐团二胡首席邓超伦共同演奏。

整场演出以其娴熟的技巧和深厚的造诣赢得了全场观众的热烈掌声。据了解，清华附中民族乐团自2002年成立至今，已经发展成为一支声部建制较为齐全、演出风格多样的民族管弦乐团。2008年4月，乐团被北京市教委授予“金帆乐团”称号。时至今日，清华附中金帆民乐团又受邀登陆国家大剧院，足以显示出乐团在长期发展中的骄人成绩。乐团秉承“自强不息，厚德载物”的民族精神，用民族音乐的优美旋律，将中华民族的高尚情操不断传承，为中华文化的发扬光大贡献力量。
