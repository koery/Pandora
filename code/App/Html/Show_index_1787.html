CCTV中学生频道《说实在的》栏目走进济宁育才中学

发布于：2013年12月21日 已被阅读次

近日，CCTV中学生频道制作人李欣博携“说实在的”和“从心开始”两个栏目组来到济宁育才中学录制节目，这是该制作组在山东省的首站录制。济宁育才中学学校的领导对此次活动高度重视，从2009级、2010级两个年级部共遴选出五位主持人、十位嘉宾和一百余名观众参与节目录制。


学生嘉宾马颍喆、马珣、戈敏在节目录制中侃侃而谈


济宁育才中学校长乔德金（右三）、副校长孙玉海（右一）、教研处主任张协成（左二）与主创人员合影


济宁育才中学教师杨昌文参与校园短剧拍摄

上午8:00，节目录制正式开始。育才中学校长乔德金、副校长孙玉海亲临现场，观看节目录制。整个录制过程历时八个多小时，共录制了“纠结的迟到”“时尚人物”“谁动了我的书包”“校园风云榜”“我是个迷茫的爪机党”等五期话题节目。录制现场主持人、嘉宾、观众面对镜头侃侃而谈，配合默契，五期专题节目一次性通过。另外，“从心开始”拍摄组也在育才中学开拍了系列短剧之《神秘的“黑手”》和《出国回来》，艾笑、李节浩等同学参与了此次短片的拍摄。
