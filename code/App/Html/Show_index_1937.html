中央电视台中学生频道在新疆地区开播暨新疆通联工作站建成

发布于：2013年12月29日 已被阅读次

10月24日上午，“CCTV中学生频道新疆开播暨新疆通联工作站”建成仪式在乌鲁木齐举行。中央电视台中学生频道总编辑郭滢、公共关系总监周华、新疆通联工作站负责人李平，新疆广播电影电视局传媒机构管理处处长刘政、人事处处长李宁，新疆广电网络股份有限公司总经理粟皓、党委书记顾贵宝、副总经理白向阳、陈长伟、总工程师李刚等领导参与了这次活动。


新闻发布会现场


新疆广播电影电视局传媒机构管理处处长刘政、新疆广电网络股份有限公司总经理粟皓、中央电视台中学生频道总编辑郭滢

中央电视台中学生频道此次携手新疆广电网络，不仅面向全疆开通171号直播频道，而且还与新疆广电网络一同搭建了天山云电视教育互动点播平台“好学课堂”，这个平台汇集了央视中学生频道提供的初一至高三的六个年级、九门来自全国的名校名师讲授的视频核心课程，新疆地区用户通过电视机、PC机、平板电脑、手机等多种终端，就可充分享受到这些优质教育资源、真正实现把优质教育资源带回家。同时，全疆中学生通过新疆广电网络的各地区营业厅，也可购买CCTV中学生频道配套学习网站www.studytv.cn的好学卡，享受初一到高三六个年级九门功课的全部优质课程。


新疆广播电影电视局人事处处长李宁、新疆广电网络股份有限公司党委书记顾贵宝、中央电视台中学生频道公共关系总监周华、新疆通联工作站负责人李平

新疆广电网络股份有限公司总经理粟皓表示：在新疆广电网络荣获“全国文化体制改革工作先进单位”称号这个契机，我们与中央电视台中学生频道进行战略合作，并引进了那么多优质教育资源，实际上是响应政府号召，为全疆基础教育事业做些实事的具体表现，也是落实新疆双语教学、教育信息化工作的一次有益尝试。相信未来，电视的智能化发展一定会给新疆的教育事业带来勃勃生机，并使得边疆教育事业蒸蒸日上。


中央电视台中学生频道新疆通联工作站负责人李平，新疆广电网络股份有限公司副总经理陈长伟为新疆通联工作站揭牌

中央电视台中学生频道总编辑郭滢表示：中学生频道在新疆的开播以及新疆通联工作站的建立，对于新疆教育界来说是件好事，这将成为新疆教育界“走出去，引进来”的一个窗口，而“好学课堂”的打造也会为新疆的中学生及教师带来实惠，通过优质教育资源的互动点播，必将提升新疆的基础教育水平，使得新疆的基础教育事业蓬勃发展，这也是落实国家教育信息化工作的一个具体尝试，未来我们还将在更多的地区建立“好学课堂”，力争让更多的师生受惠。
