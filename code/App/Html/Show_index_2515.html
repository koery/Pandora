央视优质的教育资源走进普通百姓家-- CCTV中学生频道近期举办系列巡回公益讲座

发布于：2014年01月06日 已被阅读次
各地联络站：

为了全面贯彻落实党的十八大精神，实践努力争办“人民满意的教育”的宗旨，扩大CCTV中学生频道的知名度，提高频道在各地的收视率，频道从2013年9月开始，要求各联络站在当地组织实施全面关注中学生身心健康的系列巡回公益活动，建议活动采用下面两种形式进行：1、邀请家庭教育专家进入学校面对家长进行专题讲座，内容以中学生心灵成长、中学生素质教育、中学生学习技巧与方法、中学生营养健康、中学生行为习惯等五大方面为主。2、甄选部分有代表性的家庭，邀请教育专家举办专题讲座，针对中学生学习兴趣、学习动力、学习压力等典型问题做出分析与讲解，给出相应的解决思路；邀请顾问人员与到场家庭一对一交流，依据家长和学生的个案情况，接受咨询，给予建议和指导。

为了确保此次活动的顺利进行，请各联络站做好以下工作：

一、请各联络站主动联系学校，争取各学校的大力支持，尽可能的让学校为本次活动安排各年级的家长会。

二、本年度所有讲座全公益性质，任何联络站不得以任何形式向学校、学生、家长收取任何费用。

三、本此活动邀请的主讲专家，由各联络站具体负责落实。专家基本要求：拥有专着或专述并有丰富讲座经验；具有从事家庭教育方面的实践经验；为得到当地学校更多认同，原则上聘请当地知名专家。请于2013年11月1日前将公益讲座教育专家的资料以电子邮件方式发到频道办公室，待审批通过后再做邀请。

四、请各联络站在每月20日前将下月公益活动的时间与内容安排表，以电子邮件方式发到频道办公室，待审批通过后再组织实施。

五、请各联络站将到场参加活动的学生所在学校、班级、姓名、家长的联系方式做好登记，形成登记表，初步调查学生素质，为频道节目遴选嘉宾和录制现场的观众贮备资源；活动现场还应录音、拍照；若在学校内开展的公益讲座，登记表还需学校德育处盖章确认，附上学校本次活动的负责人的联系方式。在每月5日前将以上资料以电子邮件方式发送到频道办公室，以便频道电话抽查，确认活动开展情况及评估活动开展效果。
