CCTV中学生频道英语益智节目《爱拼才会赢》即将推出

发布于：2013年12月29日 已被阅读次

《爱拼才会赢》（You Can if You Try）是CCTV中学生频道面向全国青少年首创的一档英语益智节目，节目内容形态紧贴频道定位，在突出电视节目的娱乐性、益智性、参与性和观赏性的同时，生动展现地道英语的魅力和学习英语的乐趣，不仅使参与节目的选手受益匪浅，同时也使观众从中受到启发，学到技巧。更为青少年搭建了一个增强学习动力、交流学习方法、展示学习成果、提升学习能力、树立学习榜样的舞台。《爱拼才会赢》节目为了激发青少年学习英语的兴趣和热情，特设置了丰厚的奖项，获奖选手将有机会免费赴台湾参加海峡两岸青少年文化交流、海外游学夏令营、国内夏令营等活动和更多丰厚大奖。


节目环节设置：
锋芒初露：30名选手参加。屏幕上呈现画面场景，选手按规定要求作答。第二轮：选手现场听英文歌曲，把听到的歌曲中的单词写下来。两轮按正确率给分，进行淘汰，前十五名选手晋级。
小试牛刀：15名晋级选手参加。大屏幕给出关于英语的选择题或是连线题，选手在限定时间内写出答案。以累计分数高低淘汰选手，6名选手晋级。
心有灵犀：6名晋级选手参加。抽签分组，分三组，每组两名选手。根据大屏幕上出现的英文单词，一名选手用英文将单词的意思表达出来，另一名选手猜答案。评委根据每个选手的表现给出相应的分值，3名选手晋级。
巅峰对决：3名晋级选手参加。第一轮即兴演讲，根据抽取到的英文单词，需要选手编一段英文故事或是一段演讲。第二轮选择风险题，答对加分，答错扣分，最终决出冠军、亚军和季军。(小低组此环节为才艺展示)
节目组别设置：小低组（1-3年级）、小高组（4-6年级）、初中组、高中组
节目播出时间：首播：首播周五18:30-19:30
