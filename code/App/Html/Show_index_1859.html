2012年高考今起网上报名 严审加分考生信息

发布于：2013年12月21日 已被阅读次

今早8时起，2012年高考(微博)网报通道正式开启，本市考生可登录市考试院网站www.bjeea.edu.cn或www.bjeea.cn报名交费。由于应届生源的减少，预计明年本市高考考生规模继续萎缩，考大学难度有望继续降低。

应届生无需填写基本信息

由于近年来考生数量持续下降，本市高考网报时间已由过去的一周缩短为5天。明年高考网报时段为今日8时至5日18时。“时间足够充裕。”市高招办主任高福勤说。

2010年起，本市电子学籍管理系统(CMIS)已与高考报名对接，在京就读的应届考生个人基本信息已提前录入报名系统，考生凭借学籍号以及由出生年月组成的初始密码，即可登录“个人网页”，无需再次填写基本信息，只需核对和修改。往届生则需自己填写基本信息。市高招办提醒考生需认真核对填写每一项信息，确保准确无误。

虽然报名时间充裕，步骤简便，但市高招办依然期望考生赶早不赶晚，避免遭遇报名高峰，无法顺利网报。

昨天，本市大部分高中校均已对高三学生进行了相关培训，提醒考生“避开高峰”和“管好密码”。东直门中学等学校还让每个学生对照报名流程，纸上模拟报名。往年高考报名，很多考生都习惯第一天先注册，过几天再填报，再加上考生白天上课，只能晚上或周末填报，因此预计今天以及本周末将是报名高峰，考生需合理安排时间，尽快报名。

由于考生报名时设置的密码，在明年5月填报志愿时仍需使用，而且，该密码也是考生个人信息安全的唯一保险，因此考生需牢记并妥善保管密码。往年高考，经常出现考生忘记密码，甚至是密码泄露的现象。

现场确认也可报名交费

考生在完成网上报名后，还需网上交费并现场确认，才算真正完成报名，具有参加明年高考的资格。

高考报名网上交费虽然已实施了三年，但每年都会有考生遇到“小麻烦”，为了给考生提供方便，网报时未能顺利交费的考生，在现场确认时也可交费。同时，家中无上网条件错过网报的考生也可现场报名。

明年高考，网上交费可选择的银行多达22家，考生可自由选择。高考报名费按不同的考试形式有不同的标准。高考文化课科目语文、数学、外语、文科综合或理科综合，每科报名费25元，外语口试报名费25元，艺术特长生统测每项100元，体育专业加试和美术类专业统测各为100元。如果一个考生参加全部统考科目考试，同时又参加艺术特长生测试、体育专业加试、美术类专业统测、外语口试，需要交纳的报名费将达到425元。

特殊测试别盲目报考

希望参加艺术特长生统测、外语口试、美术类专业统测、体育专业加试等特殊测试的考生，也需在报名同时交报名费。

由于外语口试提前到填报志愿前进行，为了填报志愿不受限，近年来外语口试几乎成了考生的必选项。此外，为了给考上大学多上几道保险，考生近年还喜欢多选择几项特殊测试。今年高考虽然考生人数下降，但外语口试和艺术特长生测试规模分别达到5万余人和2600余人，不降反增。

市高招办提醒考生，报考特殊测试一定要依据个人实际情况，如果盲目报考，不仅浪费考生自己的复习时间和报名费，同时也浪费了全市组考成本和资源。

拟加分考生提前准备材料

虽然本市明年高考加分照顾政策尚未公布，但考生如果想享受加分照顾，就需在报名时提前提交符合相关录取照顾条件的个人信息，高考网报结束后，考生的加分照顾资格信息将与报名信息一并封存，任何人不能更改。在市高招办看来，提前锁定考生加分信息，将避免部分考生在获知加分政策后更改身份，骗取加分照顾。

最近几天，本市部分高中均提醒符合相关照顾条件的考生抓紧时间办理各种证明材料，以便审核。明年高考，本市对加分考生信息的审核将更加严格，现场确认时，相关考生就需提前向区县高招办提供证明材料，进行审核确认。

报名时，考生还需提前对参加本科录取还是专科录取做出选择，如果考生只参加专科录取，报名时就可以直接选择“只参加语文、数学、外语3个科目的考试”。选择此项的考生将不能参加文科综合或理科综合考试，这意味着无论高考成绩如何，都将失去参与本科录取的机会。如果考生不选该项，不仅可以参加所有四科考试，即便成绩达不到本科线，依然可以参与专科录取。
